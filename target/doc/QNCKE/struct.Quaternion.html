Expand description

An element of the quaternion ring (Z/pZ)[i, j, k], i.e. a + b*i + c*j + d*k with all coefficients reduced modulo p.

Quaternions over a finite field form a non-commutative ring under the standard Hamilton multiplication rules (reduced mod p), which makes them a convenient platform group for Anshel–Anshel–Goldfeld-style protocols: multiplication is cheap, but the underlying word/conjugacy problems are non-trivial.

pub fn new(a: i128, b: i128, c: i128, d: i128, p: u64) -> Self

Create a new Quaternion, reducing all components modulo p.

Components are accepted as i128 so that negative intermediate values (e.g. from subtraction or conjugation) can be passed directly and correctly reduced into the range [0, p) via rem_euclid.

§Examples
let q = Quaternion::new(-1, 5, 0, 0, 7);
assert_eq!(q.a, 6); // -1 mod 7 == 6
assert_eq!(q.b, 5);

pub fn conjugate(&self) -> Self

Algebraic conjugate: q* = a - b*i - c*j - d*k (mod p).

This is the classical quaternion conjugate (negate the vector part), not to be confused with Quaternion::conjugate_by, which performs group-theoretic conjugation w^-1 * q * w.

§Examples
let q = Quaternion::new(1, 2, 3, 4, 11);
let conj = q.conjugate();
assert_eq!(conj.to_tuple(), (1, 9, 8, 7)); // -2,-3,-4 mod 11
Source

pub fn norm(&self) -> u64

Multiplicative norm: N(q) = a^2 + b^2 + c^2 + d^2 (mod p).

The norm is multiplicative (N(q1 * q2) = N(q1) * N(q2) mod p) and is used to compute the multiplicative inverse of q.

§Examples
let q = Quaternion::new(1, 2, 0, 0, 11);
assert_eq!(q.norm(), 5); // 1^2 + 2^2 = 5
Source

pub fn is_invertible(&self) -> bool

Check whether this quaternion is invertible modulo p.

A quaternion is invertible exactly when its norm is non-zero modulo p.

§Examples
let zero = Quaternion::new(0, 0, 0, 0, 11);
assert!(!zero.is_invertible());
Source

pub fn inverse(&self) -> Self

Compute the multiplicative modular inverse: q^-1 = q* * N(q)^-1 (mod p).

Requires p to be prime and N(q) != 0 (mod p); the norm’s modular inverse is computed via Fermat’s little theorem (N(q)^(p-2) mod p).

§Panics

Panics if the quaternion has zero norm modulo p (i.e. it is not invertible).

§Examples
let q = Quaternion::new(1, 2, 0, 0, 11);
let inv = q.inverse();
assert_eq!(q * inv, Quaternion::identity(11));
Source

pub fn conjugate_by(&self, w: &Quaternion) -> Self

Group-theoretic conjugation of self by w: computes w^-1 * self * w.

This is the operation underlying the AAG protocol’s exchange step, distinct from the algebraic conjugate (vector-part negation).

§Examples
let q = Quaternion::new(1, 2, 3, 4, 11);
let w = Quaternion::new(2, 0, 1, 0, 11);
let conjugated = q.conjugate_by(&w);
assert_eq!(conjugated, w.inverse() * q * w);
Source

pub fn identity(p: u64) -> Self

The multiplicative identity quaternion 1 + 0i + 0j + 0k over Z/pZ.

§Examples
let id = Quaternion::identity(11);
assert_eq!(id.to_tuple(), (1, 0, 0, 0));
