Expand description

The full output of a completed (or attempted) AAG key exchange between Alice and Bob.

Fields§

§alice_pattern: Vec<usize>

Alice’s private pattern (revealed here for testing/debugging only; in a real protocol this would never leave Alice’s side).

§bob_pattern: Vec<usize>

Bob’s private pattern (likewise, for testing/debugging only).

§K_alice: Quaternion

The commutator value Alice derives.

§K_bob_raw: Quaternion

The raw (pre-normalization) commutator value Bob derives.

§K_bob_normalized: Quaternion

Bob’s value after the public normalization step (inversion), which should equal K_alice on success.

§keys_match: bool

Whether K_alice == K_bob_normalized, i.e. whether the exchange succeeded.

§session_key_hex: Option<String>

The derived session key (hex-encoded SHA3-256 digest), present only if keys_match is true.
