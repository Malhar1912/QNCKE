Function bytes_to_bits

pub fn bytes_to_bits(data: &[u8]) -> Vec<u8> ⓘ
Expand description

Expand a byte slice into its individual bits (0 or 1), most-significant bit first within each byte.

§Examples

let bits = bytes_to_bits(&[0b1010_0000]);
assert_eq!(bits, vec![1, 0, 1, 0, 0, 0, 0, 0]);
